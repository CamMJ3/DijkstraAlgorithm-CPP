\doxysubsection{Detailed Description}
Se define la estructura de un grafo pesado para la implementación del algoritmo de Dijkstra. 

\begin{DoxyAuthor}{Author}
Camila Andrea Jaimes Martínez.
\end{DoxyAuthor}
Utilizando una matriz de adyacencia para representar el grafo, la plantilla de clase \doxylink{class_grafo}{Grafo} permite trabajar con diferentes tipos de peso, sean enteros o decimales. A partir de los vertices, el algoritmo de Dijkstra se especializa en calcular la distancia mínima desde un vértice "{}inicio a los demás vértices del grafo.
