\doxysubsection{Detailed Description}
El programa completo de la clase \doxylink{class_grafo}{Grafo} y el algoritmo de Dijkstra. 

Las opciones son las siguientes:\+
\begin{DoxyItemize}
\item 1) Crear un grafo con un número determinado de vértices.
\item 2) Agregar aristar ponderadas, dirigidas o no, entre dos vértices del grafo.
\item 3) Calcular y visualzar la distancia mínima desde un vértice de origen a todos los demás.
\item 4) Salir del programa. 
\end{DoxyItemize}
