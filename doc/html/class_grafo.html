template<typename T>
Grafo< T >::Grafo ( int v)

Inicializa un grafo de n vértices con una matriz de adyacencia, por lo tanto, vacía.

A partir del número de vértices, se re-dimensiona un grafo del mismo tamaño cuya matriz de adyacencia (matriz_adj) empieza en ceros, lo cual indica que no hay aristas existentes entre los vertices.

template<typename T>
void Grafo< T >::dijkstra ( int inicio)

Implementa el algoritmo de Dijkstra para calcular la distancia mínima desde un vértice de origen a todos los demás vértices del grafo.

Primero, se inicializa un vector de distancias con el valor máximo para el tipo de dato T, además de un vector de visitados en "false" para indicar que no han sido visitados. La distancia del vértice de inicio consigo mismo es de 0.

Se itera sobre los vértices del grafo, encontrado el vértice no visitado con la distancia mínima acumulada, marcándolo como visitado y actualizando las distancias de sus vértices adyacentes. Si se encuentra un camino más corto a través del vértice actual, se actualiza la distancia en el vector de distancias.
