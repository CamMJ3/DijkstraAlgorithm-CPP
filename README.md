# Determinación de circuitos en un grafo (C++)

Implementación de una clase Grafo para modelar redes de datos y encontrar los caminos más cortos entre sus vertices, sea el grafo dirigido o no dirigido, haciendo uso del algoritmo de Dijkstra.

## Características
 
- Menú interactivo.
- Inicialización de grafo con n vértices.
- Uso de una matriz de adyacencia.
- Inserción de aristas.
- Visualización del camino más corto desde un vértice de inicio a los demás vértices del grafo.
